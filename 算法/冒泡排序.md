## 冒泡排序

从第一个元素开始比较相邻的两个元素，大的往后排

```js
  function bubbleSort(arr) {
    for (var i = 0; i < arr.length; i++) {
      for (var j = 0; j < arr.length - i - 1; j++) {
        if (arr[j] > arr[j + 1]) {
          var tmp = arr[j + 1]
          arr[j + 1] = arr[j]
          arr[j] = tmp
        }
      }
    }
    return arr
  }
```
